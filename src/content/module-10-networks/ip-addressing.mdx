## Hook — The Internet's Address System

Every device on the internet has an IP address — a logical address used to route packets across networks. Unlike MAC addresses (local and fixed), IP addresses are hierarchical and enable the global routing system that connects billions of devices.

## IPv4 Addresses

```
32 bits, written as 4 octets: 192.168.1.100
Binary: 11000000.10101000.00000001.01100100

Total addresses: 2³² = ~4.3 billion (not enough!)

Classes (historical, now replaced by CIDR):
  Class A: 0.0.0.0 – 127.255.255.255    /8   (16M hosts each)
  Class B: 128.0.0.0 – 191.255.255.255  /16  (65K hosts each)
  Class C: 192.0.0.0 – 223.255.255.255  /24  (254 hosts each)
  Class D: 224.0.0.0 – 239.255.255.255  Multicast
  Class E: 240.0.0.0 – 255.255.255.255  Reserved
```

### Subnetting

```
Network address + Host address:
  192.168.1.100/24
  └───────┬───┘ └┘
  Network part  Host part

Subnet Mask: 255.255.255.0 = /24
  IP:   11000000.10101000.00000001.01100100
  Mask: 11111111.11111111.11111111.00000000
  AND:  11000000.10101000.00000001.00000000 = 192.168.1.0 (network)

/24 → 256 addresses, 254 usable hosts
  .0 = network address, .255 = broadcast

Splitting a /24 into two /25s:
  192.168.1.0/25  → hosts .1 to .126  (126 hosts)
  192.168.1.128/25 → hosts .129 to .254 (126 hosts)
```

### Private vs Public Addresses

```
Private (not routable on internet):
  10.0.0.0/8        (16M addresses)
  172.16.0.0/12     (1M addresses)
  192.168.0.0/16    (65K addresses)

Your home network uses private IPs.
Your router uses NAT to share one public IP.

NAT (Network Address Translation):
  Internal: 192.168.1.100:5000 → Router → External: 203.0.113.1:40000
  Router rewrites source IP/port for outgoing packets.
  Router maintains translation table for return traffic.
```

## See It Work — IPv6

```
128 bits: 2¹²⁸ = 340 undecillion addresses
  (enough for every atom on Earth's surface)

Format: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
Short:  2001:db8:85a3::8a2e:370:7334

Why IPv6?
  - IPv4 exhausted (~4.3B addresses for 8B+ people + IoT)
  - No NAT needed (every device gets a public address)
  - Built-in IPsec (encryption)
  - Simplified header (faster routing)

Adoption: ~40% of internet traffic (2024), growing slowly.
Dual-stack: Most devices run both IPv4 and IPv6.
```

## Why It Matters

- IP addressing is fundamental to how the internet works — every connection uses it.
- Subnetting skills are essential for network administration and cloud infrastructure.
- Understanding NAT explains why you can't easily host a server at home and how VPNs work.
